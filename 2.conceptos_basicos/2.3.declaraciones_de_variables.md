# Declaraciones de variables

```js
var // Declara una variable, inicializando opcionalmente un valor. 
let // ** Declara una variable local en un bloque de ámbito, inicializando opcionalmente un valor.
const // ** Declara una constante de sólo lectura en un bloque de ámbito.


// ** Notación válida en ECMASCRIPT 6+
```
