# Sentencia for of

La sentencia for...of crea un bucle iterando sobre objetos iterables (incluyendo Array, Map, Set, argumentos objetos etc), invocando una iteraci√≥n personalizada conectando con sentencias para ser ejecutadas por el valor de cada propiedad distinta.

```js
for (variable de objeto) { sentencias }
```

## Ejemplo

```js
let arr = [3, 5, 7];
arr.foo = "hello";

for (let i in arr) {
   console.log(i); // logs "0", "1", "2", "foo"
}

for (let i of arr) {
   console.log(i); // logs "3", "5", "7"
}
```
